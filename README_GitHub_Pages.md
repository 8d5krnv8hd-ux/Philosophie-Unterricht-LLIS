# Philosophie 4 – Positionenvergleiche & Fachwissen

Interaktive Website für den Philosophieunterricht der Schola Europaea (T. Retter).

## Seiten

| Seite | Inhalt |
|---|---|
| `index.html` | 🎴 Philosophen-Karten — alle Positionen mit Filter, Querverweisen |
| `vergleiche.html` | 📊 Positionenvergleiche — alle Tabellen, Vergleichsmodus, Quiz |
| `fachwissen.html` | 📖 Fachwissen — vollständige Philosophen-Texte aus dem Unterrichtsmaterial |
| `begriffe.html` | 📚 Kernbegriffe — alphabetisches Glossar mit Definitionen und Querverweisen |

## GitHub Pages Setup (einmalig, 3 Minuten)

1. **Repository erstellen**
   - Gehe zu [github.com](https://github.com) → «New repository»
   - Name: `philo4` (oder beliebig)
   - Sichtbarkeit: Public oder Private (Pages funktioniert bei beiden)
   - Erstellen ohne README

2. **Dateien hochladen**
   - Alle 4 HTML-Dateien per Drag & Drop in das Repository laden
   - Commit-Nachricht: «Initial upload»

3. **GitHub Pages aktivieren**
   - Repository → Settings → Pages
   - Source: «Deploy from a branch»
   - Branch: `main`, Ordner: `/ (root)`
   - Save

4. **Fertig** — nach ~1 Minute erreichbar unter:
   `https://DEIN-USERNAME.github.io/philo4/`

## Querverweise

- Karten-Seite → Vergleichstabelle: Klick auf «📊 Vergleich» in jeder Karte
- Karten-Seite → Fachwissen: Klick auf «📖 Fachwissen» in jeder Karte  
- Fachwissen → Begriffe: Klick auf Kernbegriffe unter jedem Eintrag
- Begriffe → Fachwissen: Klick auf «📖 Fachwissen» bei jedem Begriff
